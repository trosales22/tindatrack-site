# TindaTrack Site

TindaTrack (short for “Tindahan Tracker”)
TindaTrack helps small businesses track inventory and sales —so you can spend less time guessing and more time growing.

## Stack

- React (TypeScript)
- TailwindCSS
- DaisyUI
- Tanstack Query
- React Hook Form
- Zod
- Zustand

## Setup

- Run `npm install`
- Adjust the ff. in .env

```bash
  VITE_WS_BASE_URL=
  VITE_PORTAL_URL=
```

- Execute the ff. to run:

```bash
npm run dev
```
